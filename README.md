# hackathon_alumni

Projet réalisé dans le cadre d'un hackathon de 48H.

Le défi :
        - mettre en avant des projets 
        - mettre en avant des éleves
        
les conditions : un CRUD fonctionnel et une selection possible par techno.


Prerequisites

    NodeJS
    MongoDB

Installation

git clone 
cd 
npm i

Development

node server.js
